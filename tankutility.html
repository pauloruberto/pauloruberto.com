PLUGIN DEVELOPMENT
OVERVIEW

homebridge-tank-utility is a plugin for HomeBridge, the popular NodeJS server which ties unsuported devices into Apple's HomeKit protocol. This plugin integrates TankUtility devices into HomeKit. The propane tank monitors are exposed to HomeKit as humidity sensors, in order to get a percentage read out. Ideally as Apple expands the HomeKit Accessory Protocol specification, native support for tank levels will be added.

MY ROLE

For some time, I've wanted to dive into creating a HomeBridge plugin. I had previously been using HomeBridge to integrate smart devices within my own home, but they already had existing plugins. When I picked up a TankUtility monitor and saw that a plugin did not yet exist, it was time for me to dip my toes into plugin development world.
